# Aurora Perfume Store

A simple responsive perfume store website built with **HTML, CSS, and Bootstrap **.

## 🌐 Project Description

Aurora is a small e-commerce style website that showcases a collection of luxury perfumes.
The website is fully responsive and designed using Bootstrap components such as **Navbar, Grid system, and Cards**.

🌐 Live Demo
[View the Website](https://fatitab-dev.github.io/aurora-store-bootstrap/)
## 🚀 Features

* Responsive design
* Bootstrap  layout
* Product cards
* Perfume images and descriptions
* Add to Cart buttons
* Hero section with background image

## 🛠️ Technologies Used

* HTML5
* CSS3
* Bootstrap
