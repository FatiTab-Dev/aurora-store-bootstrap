# Aurora Perfume Store

A simple responsive perfume store website built with **HTML, Sass, Bootstrap 5, and jQuery**.

## 🌐 Project Description

Aurora is a small e-commerce style website that showcases a collection of luxury perfumes. 
The website is fully responsive and features dynamic interactions such as a **Dark Mode** and **Add to Cart** logic.

[View the Live Demo](https://fatitab-dev.github.io/aurora-store-bootstrap/)

## 🚀 Features

* **Responsive Design:** Fully mobile-friendly layout using Bootstrap Grid.
* **Sass-Powered Styles:** Clean and modular CSS using Sass variables and nesting.
* **Interactive UI:** Smooth effects and cart management with jQuery.
* **Dark Mode:** User-friendly theme toggle.
* **Product Showcase:** Beautifully designed product cards with hover effects.

## 🛠️ Technologies Used

* **HTML5**
* **Sass (SCSS)** - For advanced and organized styling.
* **Bootstrap 5** - For layout and responsive components.
* **jQuery** - For DOM manipulation and animations.
